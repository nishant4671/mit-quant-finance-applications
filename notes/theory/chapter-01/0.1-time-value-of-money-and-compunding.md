> "A rupee today is worth more than the same rupee tomorrow."

This is the **first principle of finance**.

Everything in finance ultimately asks one question:

> **"What is money at one point in time worth at another point in time?"**

Every major field depends on this idea:

- Bond Pricing
- Stock Valuation
- Portfolio Management
- Corporate Finance
- Banking
- Derivatives
- Fixed Income
- Quantitative Finance
- Option Pricing
- Risk Management

Without understanding Time Value of Money (TVM), none of the later topics make sense.

# 1. Definition of Time Value of Money

## Definition

The **Time Value of Money (TVM)** states that:

> **Money available today is worth more than the same amount of money received in the future because today's money can be invested to earn returns.**

In simple words,

Money has earning power.

If you own money today,

you can

- invest it,
- earn interest,
- reinvest the interest,
- and create even more money.

Therefore,

Current Money

>

Future Money

---

# 2. Why Does Money Have Time Value?

There are four major reasons.

---

## A. Investment Opportunity

Money today can earn returns.

Example

You receive

₹1000 today.

You invest it at

10%

One year later

₹1000

↓

₹1100

If someone instead promises to give you ₹1000 after one year,

you lose ₹100 of potential earnings.

---

## B. Inflation

Inflation reduces purchasing power.

Example

Today

₹100 buys

10 chocolates.

One year later

the same ₹100

may buy

only

9 chocolates.

Future money buys less.

---

## C. Risk

Future payments are uncertain.

Questions include:

- Will the borrower repay?
- Will the company survive?
- Will payment be delayed?
- Will inflation rise?

Today's money has no future uncertainty.

---

## D. Liquidity

Money today can immediately be

- spent,
- invested,
- transferred,
- saved.

Future money cannot.

---

# Core Principle

Money

↓

Time

↓

Value Changes

Time itself changes the value of money.

# 3. Present Value (PV)

## Definition

Present Value is

the value today

of money received in the future.

It answers:

> "What is future money worth today?"

---

# Present Value Formula (Discrete)

```
           FV
PV = ----------------
      (1+r)^t
```

Where

PV

Present Value

FV

Future Value

r

Interest Rate

t

Number of Years

---

## Example

Future Value

₹1100

Interest

10%

Time

1 Year

```
          1100
PV = ------------
         1.10

PV = ₹1000
```

Interpretation

Receiving

₹1100

one year later

is financially equivalent to

receiving

₹1000

today

if money earns

10%.

---

# 4. Future Value (FV)

## Definition

Future Value is

the amount that today's money grows into after earning interest.

Question answered

> "If I invest today, how much money will I have later?"

---

# Future Value Formula (Discrete)

```
FV = PV(1+r)^t
```

---

Example

Present Value

₹5000

Interest

8%

Time

3 Years

```
FV

=5000(1.08)^3

=₹6298.56
```

Meaning

₹5000 invested today

becomes

₹6298.56

after three years.

---

# Relationship Between PV and FV

Present Value

↓

Compounding

↓

Future Value

Future Value

↓

Discounting

↓

Present Value

Compounding and Discounting

are exact mathematical opposites.

# 6. Simple Interest

## Definition

Interest is calculated

ONLY

on the original principal.

---

# Formula

```
Interest

=

Principal × Rate × Time
```

or

```
I = Prt
```

Final Amount

```
A = P(1+rt)
```

---

Example

Principal

₹1000

Interest

10%

Time

3 Years

```
Interest

=1000×0.10×3

=₹300

Final Amount

₹1300
```

---

Characteristics

✔ Constant growth

✔ Linear growth

✔ Interest never earns interest

---

# 7. Compound Interest

## Definition

Interest is calculated on

- Original Principal
- Previous Interest

This is called

Interest-on-Interest.

---

# Formula

```
FV = PV(1+r)^t
```

More generally,

```
A = P(1+r)^t
```

---

Example

₹1000

10%

3 Years

Year 1

1000

↓

1100

Year 2

1100

↓

1210

Year 3

1210

↓

1331

---

Simple Interest

₹1300

Compound Interest

₹1331

Compound interest always grows faster.

# 8. Discrete Compounding

## Definition

Interest is added

at fixed intervals.

Examples

- Yearly
- Half-Yearly
- Quarterly
- Monthly
- Weekly
- Daily

---

General Formula

```
                 nt
FV = PV(1+r/n)
```

where

```
           nt
FV = PV(1+r/n)
```

Mathematically,

```
           nt
FV = PV(1+r/n)
```

Where

PV

Present Value

r

Annual Interest Rate

n

Compounding Periods Per Year

t

Years

---

Examples

Yearly

n=1

Monthly

n=12

Daily

n=365

---

Example

Principal

₹1000

Interest

12%

Monthly

2 Years

```
              12×2
FV=1000(1+0.12/12)

=₹1269.73
```

---

# Effective Annual Rate (EAR)

Different compounding frequencies

produce different effective returns.

Formula

```
              n
EAR=(1+r/n)-1
```

Actually,

```
                n
EAR = (1+r/n)^n −1
```

---

Example

Nominal Rate

12%

Monthly

```
EAR

=(1+0.12/12)^12−1

≈12.68%
```

Meaning

Although the bank advertises

12%

you actually earn

12.68%.

# 9. Continuous Compounding

## Definition

Instead of compounding

monthly

or

daily,

interest is added

continuously,

at every infinitesimally small instant.

There are infinitely many compounding periods.

---

As

```
n → ∞
```

Discrete Compounding

becomes

Continuous Compounding.

---

# Mathematical Derivation

Start with

```
              nt
FV=PV(1+r/n)
```

Take

```
n→∞
```

Then

```
              rt
FV=PVe
```

This is one of the most important limits in mathematics.

```
          n
lim(1+1/n)=e
n→∞
```

Euler proved

```
e≈2.718281828459...
```

---

# Continuous Compounding Formula

```
FV=PVe^(rt)
```

Where

e

Euler's Number

---

Example

PV

₹1000

r

8%

t

5 Years

```
FV

=1000e^(0.08×5)

≈₹1491.82
```

---

# Continuous Discounting

Moving

future money

back

to today.

Formula

```
PV=FVe^(-rt)
```

Notice

Negative exponent

↓

Moving backwards through time.

---

Example

FV

₹2000

r

6%

t

3 Years

```
PV

=2000e^(-0.06×3)

≈₹1670.54
```

# 10. Relationship Between Compounding Frequency

Suppose

| Compounding | Future Value |
|--------------|-------------:|
| Yearly | ₹1100.00 |

| Quarterly | ₹1103.81 |
| Monthly | ₹1104.71 |

Observation

Increasing compounding frequency

Continuous Compounding

# 11. Discount Factor

Definition

A Discount Factor converts future money

into

present money.

---

Formula

```
DF

=

1/(1+r)^t
```

Continuous Version

```
DF=e^(-rt)
```

Relationship

```
PV=FV×DF
```

---

Example

Interest

5%

Time

2 Years

```
DF

=

1/(1.05)^2

=0.90703
```

Meaning

₹1 received after two years

is worth

₹0.907

today.

---

# 12. Growth Factor

Opposite of Discount Factor.

Formula

```
GF=(1+r)^t
```

Continuous Version

```
GF=e^(rt)
```

Relationship

```
FV=PV×GF
```

# 17. One-Sentence Revision

The Time Value of Money states that money available today is worth more than the same amount in the future because it can earn returns; present value and future value are linked through discounting and compounding, discrete compounding assumes interest is added at fixed intervals, continuous compounding models growth at every instant using Euler's number **e**, and these concepts form the mathematical foundation of all modern financial valuation.
