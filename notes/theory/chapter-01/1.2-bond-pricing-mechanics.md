# 1. Present Value of a Single Cash Flow

If a future payment (FV) is received after t years at an annual interest rate r,

PV = FV / (1 + r)^t

Where

PV = Present Value

FV = Future Value (cash received later)

r = Discount rate (decimal)

t = Time in years

------------------------------------------------------------

Example

FV = ₹1000

r = 8%

t = 3 years

PV

= 1000 / (1.08)^3

= ₹793.83

------------------------------------------------------------

# 2. Present Value Using Continuous Compounding

If discounting is continuous,

PV = FV × e^(-rt)

Where

e = Euler's Number ≈ 2.718281828

------------------------------------------------------------

Example

FV = ₹1000

r = 5%

t = 2 years

PV

= 1000 × e^(-0.05×2)

≈ ₹904.84

------------------------------------------------------------

# 3. Bond Pricing Formula

The value of any bond equals the present value of every future cash flow.

General Formula

                 n
Bond Price = Σ  C/(1+r)^t  +  F/(1+r)^n
               t=1

Where

P = Bond Price

C = Coupon Payment

F = Face Value (Principal)

r = Discount Rate (YTM if using a single rate)

n = Number of periods until maturity

------------------------------------------------------------

Expanded Form

Bond Price

= C/(1+r)

+ C/(1+r)^2

+ C/(1+r)^3

+ ...

+ (C+F)/(1+r)^n

Every cash flow is discounted separately.

------------------------------------------------------------
